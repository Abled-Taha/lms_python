1. If you are not using NixOS, install "mise", if you are using NixOS, proceed to step 2.
2. Clone the repo and cd into it.
3. If you are using NixOS, then run "nix develop", if you are not using NixOS, proceed to step 4.
4. If you are on Linux, run "setup.sh", if you are on windows, run "setup.ps1".
5. Follow the instructions of the script.
6. You are all done. "mise run dev" can be used to start the dev server.
